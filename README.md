## Overview
FunkoLens is an Angular-based Progressive Web App (PWA) designed to analyze recent eBay marketplace sales data for Funko collectibles.

The application focuses on completed sales and provides pricing statistics such as averages, medians, and trimmed means, as well as margin calculations based on configurable fees and costs.

> Note: This project is intended for private and educational purposes.
> All data is sourced from publicly available marketplace information.
